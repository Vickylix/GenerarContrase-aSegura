# GenerarContraseniaSegura
es un pequeño codigo en python ver 3.14.5
#¿que hace?
es un programa por consola que le pide al usuario la longitud de una contraseña y la pc "piensa" una contraseña con letras (minusculas y mayusculas), numeros y simbolo en
proporcion de la longitud que pide el usuario.

#Como funciona
una vez descargado y ejecutado el codigo en python o en un interprete de python por consola se le pedira que ingrese una longitud para "crear" una contraseña de dicha longitud.
